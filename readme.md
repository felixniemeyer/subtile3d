# subtile3d

A computer graphics demo that runs in the browser (WebGL).

## preview .gif
[preview](subtile3d.gif)

## build

```git clone ...```

```cd subtile3d```

```npm install```

either: 
```npm run dev```

or: 
```npm run build```

## run 
build and then open ```./dist/index.html``` with a modern browser
